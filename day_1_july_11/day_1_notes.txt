Web Development
===============
-Developing website


1. FrontEnd: User Interface ( Look & Feel )
        HTML: Content ( button, inputbox, div, p, table, list )
        CSS:  Styling ( border , color , background, margin, padding )
        Bootstrap: CSS Library  ( pre-defined css classes )
        Javascript: Dynamic pages, behaviour 
        React: Javascript library 

2. BackEnd: Develop REST APIs
            urls like: https://fakestoreapi.com/products

3. DataBase: Holds the data ( oracle, mysql, mongoDB, Postgre )

    b. alt+b  /  alt+shift+b
  


How to add/include Bootstrap in any HTML File
=============================================
add CDN Link of bootstrap in HTML file using <link>
   <link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.8/dist/css/bootstrap.min.css" rel="stylesheet">




Most Commonly Used Javascript Objects
======================================
console : log(), warn(), error(), table(), time(), timeEnd()
document : write(), getElementById(), querySelector()
window : alert(), confirm(), prompt(), atob(), btoa()
Math :  max() , min()
Array : push(), filter(), reduce()
Object :  assign(), keys(), values()
JSON :  stringify(), parse()


alert() : Display Informative Message
          Message + Ok

confirm() : User Confirmation
            Message + Ok(true)    cancel(false)

prompt() : To Collect User Input 
           inputbox + message + ok,cancel
